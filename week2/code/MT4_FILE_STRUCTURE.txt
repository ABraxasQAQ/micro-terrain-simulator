MT4/MT5 文件结构说明
====================

当前主线已经分成两段：

MT4：端口识别

- 只负责确认监测点方位、baseline、全端口扫描、端口方位分析表。
- 不负责最终地形 preset。
- 不负责 MATLAB 优化。

MT5：MATLAB 多地形优化

- 读取/参考 MT4 的端口方位表。
- 选择 ActivePortIds。
- 设置目标地形 TargetZ。
- MATLAB patternsearch 优化电压。
- 保存最终 preset。


保留的 MT1 底座
--------------

功能：

- 调用相机和 opencv_wrapper。
- 读取 CurrentVoltage.dat。
- 输出 AveragePos.txt / measure_log.json。
- 应用最终 preset。

文件：

- MT1_shape_measure.py
- MT1_SinglePort_wrapped.py
- MT1_terrain_presets.py
- MT1_apply_terrain.py
- MT1_GetShape_TargetTerrain.m
- MT1_GetShapeDiff_TargetTerrain.m


保留的 MT4 端口识别文件
----------------------

功能：

- 确认点位。
- 做 baseline。
- 做全端口扫描。
- 生成端口方位表。

文件：

- MT4_check_point_mapping.py
- MT4_response_scan.py
- MT4_analyze_port_mapping.py
- MT4_MAPPING_NOTES.txt
- MT4_RUN_GUIDE.txt
- MT4_WORKFLOW_README.txt
- MT4_FILE_STRUCTURE.txt

输出：

- MT4_runs/日期_baseline/
- MT4_runs/日期_port_scan/
- MT4_outputs/port_mapping/


保留的 MT5 优化文件
------------------

功能：

- 基于 MT4 端口表选择 ActivePortIds。
- MATLAB 优化目标地形。
- 保存最终 preset。
- 记录当前编号阶段的功能。

文件：

- MT5_main_OptimizeTerrain.m
- MT5_save_optimized_preset.py
- MT5_RUN_GUIDE.txt
- MT5_STAGE_LEDGER.txt

输出：

- MT5_outputs/日期_地形名_optimize/
- MT5_outputs/final_presets/
- MT1_presets/地形名.json


已删除/废弃
-----------

- MT2 扫描测试文件。
- 旧 README_TEST_STEP。
- 早期 MT1 单点/目标地形 prototype 入口。
- 旧 MT4 优化入口。

原因：

- 扫描阶段统一归 MT4。
- 优化阶段统一归 MT5。
- 最终地形 preset 必须来自 MATLAB 优化结果，不再来自手动挑电压。
