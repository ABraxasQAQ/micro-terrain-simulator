## 二、物料、外采加工与工具清单

> 为确保项目在极短时间内顺利落地，所有依赖外部采买的非标件和活体需在**第一周内完成下单**。

---

### 2.1 物料外采与加工清单

#### 2.1.1 硬件制备耗材与加工

| 物料 / 加工项 | 用途说明 |
| :--- | :--- |
| **亚克力板材加工** | 定制激光切割亚克力板（用于浇筑薄膜的底层模具）。 |
| **定制 PCB 控制背板** | 用于液态金属通道的电压触点转接（沿用或按需打样）。 |
| **Ecoflex 硅胶（00-30）** | 用于制作高弹性的软体薄膜基底。 |
| **液态金属（Galinstan）** | 用于填充微通道，实现导电发热膨胀。 |
| **医用注射器与平头针头** | 若干，用于极细通道的液态金属注射。 |

#### 2.1.2 活体实验与 Demo 耗材

| 耗材 | 用途说明 |
| :--- | :--- |
| **实验蚂蚁** | 网购大尺寸、易识别的蚁种（如日本弓背蚁），约 30～50 只备用。 |
| **特氟龙防逃逸液（Fluon）** | 涂抹于测试平台边缘，防止蚂蚁逃脱。 |
| **高亮 LED 聚光灯** | 作为全局统一的环境诱导源（利用趋光性取代食物）。 |
| **无水酒精与无尘布** | 用于每次实验后擦拭薄膜，消除残留的生物信息素。 |
